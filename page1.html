What is GitHub

GitHub is a code hosting platform for version control and collaboration.It lets you and others work together on projects from anywhere. It’s free, easy to use, and has become central in the movement toward open-source software. It also enables developers to upload their own code files to collaborate with their fellow developers. There are millions of programmers and hobbyists that download, review, and evaluate each other's workGitHub is a for-profit company that offers a cloud-based Git repository hosting service. Essentially, it makes it a lot easier for individuals and teams to use Git for version control and collaboratation, GitHub’s interface is user-friendly enough so even novice coders can take advantage of Git. Without GitHub, using Git generally requires a bit more technical savvy and use of the command line. As a company, GitHub makes money by selling hosted private code repositories, as well as other business-focused plans that make it easier for organizations to manage team members and security. We utilize Github extensively at Kinsta to manage and develop internal projects.

History of GitHub

  • The company, GitHub, was formed in 2007 by Chris Wanstrath, P. J. Hyett, Tom Preston-Werner and Scott Chacon, and the GitHUb service was rolled out in February 2008.
  • 2009 – June 2013- GitHub continues to release products including GitHub Enterprise, Redcarpet, and Hubot. Many companies that now regularly use GitHub – including Facebook and Google – join GitHub during this period.
  • July 2013 – September 2015- GitHub continues to launch a series of products and enhancements to existing products. For the desktop, it releases Electron, Atom, and a desktop client. In terms of outreach, it launched the Bug Bounty Program, ChooseALicense.com, GitHub Classroom, GitHub Student Developer Pack, and the GitHub Engineering blog. The GitHub website also experiences multiple attacks as well as censorship from governments. In April 2014, co-founder and CEO Tom Preston-Werner resigned the company following allegations of harassment.
  • October 2015 – present- GitHub changes its pricing model from a repository-based one to a user-based one; in the process, it introduces unlimited private repositories for paying customers.
